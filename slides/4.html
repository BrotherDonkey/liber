We are working on ...

  • Moving good components into Atlas (with design and dev review)
  • Improving developer and designer facing documentation
  • Providing more full featured atomic classes.
  • Removing duplicate functionality from docs-ui.

How does an existing component get moved?

  • A discussion begins. (Github Discussions)
  • Design review of (proposed/existing) component. (Figma / Code)
  • Pull request into Atlas.
  • Documentation pull request into Atlas site.
