Getting your change into docs-ui

To do this you'll need to update and install @microsoft/atlas-css

  1. Increment the version in Atlas (See next slide for image).
  2. Install that version in docs-ui.
  3. Run `npm install` and commit the lockfile.
  4. Note: there a lag between when our new version is available on public npm and our private Devops registry 🙁. 2-6 hours!
